# Stakeholders

| Stakeholder | Interest | Engagement |
|-------------|----------|------------|
| Field sales representatives | Fast mobile access to customers and tasks | Persona, journeys, UAT |
| Sales management | Pipeline visibility, activity compliance | Reporting via Gen (not duplicate CRM DB) |
| IT / ABRA administrators | Security, Gen permissions, API limits | Integration review, OpenAPI access |
| Finance / ERP | Document integrity in Gen | Order/offer flows mapped to Gen docs |
| Product owner | MVP scope and release order | Owns `mvp-scope.md` |

## Assumptions (validate)

- Users have company-managed mobile devices or BYOD with MDM policy TBD.
- Each user maps to an ABRA Gen identity with CRM-relevant permissions.
- Training materials will reference Gen terminology (Firm, not “Account” unless UI translates).
