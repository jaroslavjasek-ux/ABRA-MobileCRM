Keďže sa ti to už stalo viackrát, odporúčam pripraviť si dva .cmd súbory.

start-backend.cmd
cd /d D:\ABRA\ABRA-MobileCRM\src\MobileCrm.Adapter
dotnet run
pause
start-frontend.cmd
cd /d D:\ABRA\ABRA-MobileCRM\src\MobileCrm.Web
npm run dev
pause
